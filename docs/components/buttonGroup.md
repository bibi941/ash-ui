# ButtonGroup(按钮组)

::: warning
ash-button-group的子元素必须全部为 ash-button
:::

###  基础用法
 <br/>
 
<ClientOnly>
  <button-group-demos-1></button-group-demos-1>
</ClientOnly>

 <br/>
 
```html
<ash-button-group>
  <ash-button type="default">上一页</ash-button>
  <ash-button type="default">确定</ash-button>
  <ash-button type="default">下一页</ash-button>
</ash-button-group>
```
